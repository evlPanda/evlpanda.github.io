You must include the following when saving your CI. If you don't it will error silently. It will appear to have worked but in effect it does nothing. If it errors it won't tell you.


      If Not (&myCI.Save()) Then
         Local any &COLL = %Session.PSMessages;
         For &i = 1 To &COLL.Count
            Local any &ERROR = &COLL.Item(&i);
            Local any &TEXT = &ERROR.Text;
            MessageBox(0, "", 0, 0, &ERROR | " " | &TEXT);
         End-For;
         &COLL.DeleteAll();
      End-If;

Also, to avoid going mad I recommend setting the following property of your CI. Things like &itmSomeTable.Count don't get updated in real time otherwise! You'll insert rows and won't see this value update until after you save.

      
    &myCI.InteractiveMode = True;
