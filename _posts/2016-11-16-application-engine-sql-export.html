To export Application Engine SQL steps as plain text you can find the SQL stored in the pssqltextdefn table.

select 
  d.sqlid, d.descr
, s.sqltext
from 
  psaestmtdefn d
, pssqltextdefn s
where d.ae_applid = 'YOUR_APP_ENGINE_NAME'
and d.ae_stmt_type = 'S'
and s.sqlid = d.sqlid
;

You can then use something like a combination of Sublime Text and the SQL Beautifier package to get some sort of SQL formatting going.

Use markdown to indicate each SQL statement in your document ('#' for heading level 1).

And then use Pandoc to convert from markdown to Word's docx format.

    brew install pandoc
    pandoc -f markdown -t docx yourDocument.md -o yourDocument.docx
