I can't be sure, but I think it has to do with dot syntax.

For example as you write PeopleCode using the dot notation... is it notation or syntax? ...it looks like this:

&RowSet(&i).Record.Field.Property

.Name is the default else you'd have to write the longer and equally correct version:

&Rowset(&i).GetRecord(Record.RecordName).GetField(Field.FieldName).Property

Because .Name is the default property the shorthand version works. In the same vein .GetRecord and .GetField are the default methods of the Row and Record objects respectively. The shorthand code is just using the default methods and properties of each object in the chain.
