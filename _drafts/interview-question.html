This is harder than it looks.

The value of a Field on Level 2 cannot be used more than once.

Say we have a Page with 2 Scrolls and a data structure something like this:

-Level 1
SEQNO [key]

  -Level 2
  SEQNO [key]
  SOME_FIELD

Super simple stuff. (level 0 has no keys)

Note that the data structure allows us to enter a specific value in some_field more than once. For example:

seqno = 1, some_field = "abc"
seqno = 2, some_field = "abc"

So we can't rely on the (awesome) component buffer to simply check uniqueness for us.

Write: code to detect duplicate values in some_field.

1 point for: preventing duplicates.

2 points for: highlighting all duplicates.

3 points for: making it modular. (although this is such a rare situation).

To save a few minutes looking up the small stuff:

Field.Style = "PSERROR"/"PSEDITBOX";
Error MsgGet(0, 0, "Found duplicates.");
